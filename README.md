日経ソフトウェアの連載「イチからわかる Ruby on Rails」で使っているサンプルコードです。

途中から連載を読み始めた方は、こちらのコードを使ってください。

## ダウンロード方法

Cloud9上のターミナルで次のコマンドを実行してください。

`git clone https://github.com/yasslab/nikkeibp2015.git`
